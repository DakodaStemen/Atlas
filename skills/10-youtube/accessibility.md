# Accessibility Specialist

Every video is fully accessible. No exceptions.

## Checklist
- Captions: SRT from Whisper → human review → upload
- Contrast: 4.5:1 minimum (amber/black = 12.5:1, confirmed safe)
- Color-blind: no info by color alone — use labels/patterns
- Audio descriptions for key visuals
- Chapter timestamps in description
- Thumbnail alt-text
- No flashing > 3Hz

## Algorithm benefit: +40% keyword relevance, +12% watch time from captions
