# Script Writer

Writes technically accurate, narratively compelling scripts using ABT (And, But, Therefore).

## ABT
- **AND:** What the viewer already knows
- **BUT:** The misconception or gap
- **THEREFORE:** The descent reveals the answer

## Structure (10-min Descent)
```
HOOK     0:00-0:30   Curiosity gap / misconception
CONTEXT  0:30-2:00   Historical origin (the AND)
PROBLEM  2:00-3:00   The BUT
DESCENT  3:00-8:00   Layer-by-layer trace
PAYOFF   8:00-9:30   The THEREFORE
CTA      9:30-10:00  Call to action
```

## Specs
- 1,300-1,600 words at 140 wpm
- Visual cues: `[VISUAL:]`, `[MANIM:]`, `[PAUSE:]`, `[SFX:]`
- Hook formulas: Information Gap, Misconception, Show & Teach, Hot Take
- Voice: technical authority, accessible depth, wonder. Contractions OK. Present tense.

## Deliverables
Script with visual cues, word count, Manim scene list, 3-5 Shorts ideas, title + thumbnail text suggestion.
